Utiliza la combinación de teclas Ctrl + Command + 3 para realizar una captura de toda la pantalla.

Para capturar parte de la pantalla, tan solo usa las teclas Ctrl + Command + 4 y selecciona el área.

En caso de que solo quieras capturar una ventana o un menú, utiliza la combinación Ctrl + Command + 4 + Barra espaciadora y selecciona la ventana o menú.

Las capturas de pantalla estarán en tu escritorio.

Si quieres más información, consulta la documentación de Apple.
