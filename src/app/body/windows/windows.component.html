Puedes hacer capturas utilizando la Herramienta Recortes de Windows o Snip & Sketch (Herramienta Recortes de Windows 11).

También puedes pulsar Windows + Shift + S para abrir la Snip & Sketch directamente.

Una vez realices la captura con esta herramienta, se te copiará directamente al portapapeles.
Puedes pegarla en cualquier sitio pulsando Ctrl + V o pulsar en la notifiación para editar la captura.
